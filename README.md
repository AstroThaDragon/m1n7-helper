# Enceladus
A custom Discord bot for **The Cosmic Lair**.

## Features - WIP
* **Quick Report (`!qr`):** Allows users to quickly report messages to the staff channel.
* **Auto-Cleanup:** Deletes the report command usage to keep channels tidy.
* **More to come!**

## Setup
This bot is designed to run on Railway. You must set the `DISCORD_TOKEN` environment variable for it to function.
