Terms of Service & Privacy Policy for M1N7 Helper
1. Data Collection
To provide its services, M1N7 Helper accesses the following data through the Discord API:

Message Content: The bot reads messages to identify and process the !qr (Quick Report) command.

Member Information: The bot monitors when users join, leave, or update their status (like server boosting) to send automated welcome, goodbye, and celebration messages.

2. Data Usage
Reports: Text provided during a !qr command is shared directly with the server's staff channel for moderation purposes.

Automation: Member data is used solely to trigger real-time events, such as the welcomer system and boost announcements.

Status: The bot uses internal lists to rotate its own status every 15 minutes.

3. Data Retention
M1N7 Helper does not store message logs or member data in a long-term database.

Data is processed in real-time to execute commands and is not sold, traded, or shared with third parties.

4. User Consent
By participating in The Cosmic Lair and using bot commands, you acknowledge and consent to the bot's access to this information for the purposes listed above.****
